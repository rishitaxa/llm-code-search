A production-ready, local, LLM-powered code search engine that allows you to
search large codebases using natural language.

Built with FastAPI, FAISS, sentence-transformers, and a local LLM via Ollama.
Works fully offline after setup. No paid APIs required.

---

## Features

- Semantic (meaning-based) code search
- Vector embeddings using sentence-transformers
- Fast similarity search with FAISS
- Local LLM explanations (DeepSeek / CodeLlama)
- Minimal, modern frontend UI
- FastAPI backend
- Docker-ready
- Cross-platform (Windows / Linux / macOS)

---

## Tech Stack

Backend:
- Python 3.11
- FastAPI
- FAISS
- sentence-transformers
- Ollama (local LLM)

Frontend:
- HTML
- CSS
- Vanilla JavaScript
